# Antoine Griffard

This site is made with:

- [Astro](https://astro.build/)
- [React](https://reactjs.org/)
- [Tailwind](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## How to run locally

Install

```bash
npm install
```

Run

```bash
npm run dev
```
